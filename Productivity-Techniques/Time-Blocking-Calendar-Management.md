# Time Blocking & Calendar Management

## Overview

Using time blocking and effective calendar management to take control of your schedule.

## Time Blocking

### What is Time Blocking?
- Schedule specific blocks for tasks
- Assign time to priorities
- Protect important work
- Visual schedule

### How to Time Block
1. List all tasks
2. Estimate time needed
3. Block time in calendar
4. Include buffer time
5. Review and adjust

### Types of Time Blocks
- **Deep work blocks**: Focused work
- **Shallow work blocks**: Administrative tasks
- **Meeting blocks**: Scheduled meetings
- **Buffer blocks**: Transition time
- **Break blocks**: Rest and recovery

## Calendar Management

### Calendar Systems
- Single calendar vs. multiple
- Color coding
- Categories and labels
- Sync across devices

### Calendar Best Practices
- Block time for priorities first
- Include travel time
- Buffer between meetings
- Schedule breaks
- Review weekly

### Calendar Templates
- Weekly template
- Daily template
- Meeting templates
- Block templates

## Scheduling Strategies

### Theme Days
- Assign themes to days
- Monday: Planning
- Tuesday: Deep work
- Wednesday: Meetings
- Thursday: Creative work
- Friday: Review and wrap-up

### Batching
- Group similar tasks
- Batch meetings
- Batch email
- Batch errands
- Reduce context switching

### Time Constraints
- Parkinson's Law: Work expands to fill time
- Set shorter deadlines
- Use timers
- Create urgency
- Increase focus

## Protecting Your Time

### Saying No
- Evaluate requests
- Protect priorities
- Set boundaries
- Offer alternatives
- Be firm but polite

### Meeting Management
- Decline unnecessary meetings
- Suggest alternatives
- Set time limits
- Have clear agendas
- End on time
